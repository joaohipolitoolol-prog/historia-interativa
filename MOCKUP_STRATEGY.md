# Mockup Strategy

## Arquivos

| Arquivo | Uso |
|---|---|
| `/images/hero-bundle.webp` | Hero principal |
| `/images/premium-bundle.webp` | Bloco Premium |
| `/images/previews-grid.webp` | Seção de prévias |
| `/images/preview-01..06.webp` | Thumbnails + lightbox |

## Como trocar

1. Substitua o arquivo em `public/images/`
2. Ou altere o path em `offerConfig.ts` (`PRODUCT_IMAGE`, `PREMIUM_IMAGE`, `PREVIEWS_GRID_IMAGE`)
3. Rode `npm run build`

## Direção visual
Educacional premium, laranja/branco, volume tangível, sem pessoas genéricas.
