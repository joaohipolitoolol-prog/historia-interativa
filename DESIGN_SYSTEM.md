# Design System — História Interativa

## Direção visual

Educação + organização + energia clara. Laranja conduz; verde confirma; navy contrasta.

## Cores

| Token | Hex | Uso |
|---|---|---|
| orange | `#FF5A1F` | CTA, selos, acentos |
| orange-light | `#FF8A3D` | Hover |
| yellow | `#FFC83D` | Destaques suaves |
| green | `#10B981` | Checks, garantia |
| navy | `#172033` | Títulos, fundos de impacto |
| warm | `#FFF9F4` | Fundo principal |
| cool | `#F3F8FC` | Seções alternadas |
| ink | `#201A17` | Texto |
| muted | `#625B56` | Texto secundário |

## Tipografia

- Display/títulos: **Manrope**
- Corpo: **Plus Jakarta Sans**
- Máximo: 2 famílias

## Escala mobile

- Texto comum: 16px
- Secundário: 15px
- Card title: 18px
- Botões: 16–17px
- Headline: 36–46px

## Componentes

- Botões grandes, área de toque confortável
- Cards só quando estruturam informação (problema, tipos, bônus)
- Hero sem selos competindo; uma composição
- Sticky CTA após primeira dobra, dismissível

## Motion

- Fade-up discreto
- Respeita `prefers-reduced-motion`
- Sem shake, confete, contadores falsos
